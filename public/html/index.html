Protean

Overview

Take control of your RESTful API's. Test them, mock them, simulate errors and document how to use them. No invasive changes to your code base. Swap API behaviour out real time over the network.

Features

  • Mock API's with a portable concise JSON like language
  • Simulate error response status codes per project or per resource path
  • Configure probability of error per project or per resource path
  • Verify request structure; headers, query string params, body payload json keys, url encoded forms
  • Auto generate curl commands to test your API's
  • Automatically integration test your API simulation
  • Automatic documentation generated for all configured projects
  • Hotswap API behaviour on the fly over the network
